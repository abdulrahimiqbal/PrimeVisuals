# STANDING GOAL: the breakthrough campaign — runs until a real one lands or escalates

## Context pickup (do this first, every run)
The goal and strategy are fixed in COUNCIL.md — read it. Then load state
from the repo; each run is a fresh window, disk is the only memory:
- MACHINE_HOW_TO_USE.md — the tool interface and the AUDIT GATE.
- KNOWLEDGE.md — the compounding memory. Scan it: `OPEN` tags are your live
  lead queue; `GRAVEYARD`/`KNOWN-MATH` are settled — do NOT redo them.
- The most recent logs/*.md HANDOFF or STUCK PACK — where the last run
  stopped and its next attack.
Rank the OPEN leads by the EV order below, resume the top one. Append to
KNOWLEDGE.md as you go; before stopping, write a HANDOFF (state, best lead,
exact next attack). That write is the only continuity — no HANDOFF, no thread.

## Search order (highest expected value first)
1. TWO-UNIVERSES Sprints 2–3 (prompts/two-universes.md): Chowla decay /
   twin density vs the F_q[t] theorems, then the matched divergence battery.
   Audit-safe — a theorem calibrates the F_q[t] side. A divergence that
   clears the gate is a breakthrough.
2. UNCOMPUTED cross-statistics nobody publishes: does μ(n) know gap(n)? do
   gaps differ after quadratic residues vs non-residues? second/triple
   correlations, conditional spacing laws. Where LO-S and murmurations
   actually came from — your native edge.
3. Famous objects (ψ, M, π, twin counts) score ~0 novelty — calibration only.
Cycle per prompts/research-loop.md: pick the top lead, design the decisive
experiment and its null BEFORE computing, answer it fully, log a
universe-labeled entry with a CONNECTION line, spawn ≤2 leads, repeat.

## Breakthrough = the ONLY completion
A ⭐⭐ finding — a new replicable STATISTICAL LAW of primes (LO-S /
murmuration class) or a cross-universe SHARED LAW / DIVERGENCE — that passes
EVERY audit-gate item, in writing:
1. FACTOR CHECK — not a zero-free-multiplier twist of a catalog object (L2).
2. COMPOSITE CONTROL — if it mentions primes, test 25,35,49,77,91,143…; one
   composite satisfying it kills it (Farey). Cramér tests density, not
   primality — no substitute.
3. ≥5-seed Cramér contrast: real beats fake.
4. PERSISTENCE: sharpens at 2× and 4× range.
5. HOLDOUT: survives a disjoint range it was not found on.
6. NOVELTY: beats the nearest catalog entry on CONTENT (quote it).
Stated as a precise conjecture (quantity ± resolution, exponent, range) with
a `shot`, a link, and — if theorem-shaped — a derivation via named classical
tools or an honest CONJECTURAL label. No partial credit. Anything failing a
gate item is logged at its true grade and the campaign CONTINUES.

Self-verification has a 0/3 record here (ψ, L2, Farey each passed the
agent's own checks, each was mis-framed). Run the gate adversarially against
your OWN candidate before believing it.

## The two exits
- ⭐⭐ clears the gate → STOP, write the expert pack (object, statement,
  evidence, novelty, Lean stub).
- Leads exhausted with none → STUCK PACK: sharpest open question, what was
  tried, what an expert could answer. A robust observation's true exit is a
  human mathematician.
Honesty regime always: predeclared nulls, no all-x claims from finite ℤ
data, ranked graveyard is valid output, never mix universe grades.
